# 关于鲁能巴蜀中学
重庆市鲁能巴蜀中学校，是重庆市江北区教育委员会主管的一所公办完全中学。

学校是全国青少年校园足球特色学校、全国青少年财商教育示范学校、重庆市优质课程资源研发基地学校、重庆市基础教育校本教研示范学校、重庆市智慧校园示范学校、重庆市江北区科普教育基地。
重庆市鲁能巴蜀中学校创办于 $2007$ 年 $9$ 月，时为民办中学。$2019$ 年被评为全国青少年校园足球特色学校。$2021$ 年 $8$ 月，学校由重庆市江北区人民政府接管，转为公办中学。

截至 $2022$ 年 $5$ 月，学校占地面积 $163$ 亩，建筑面积约 $141926$ 平方米；有教职工 $700$ 余人，在职教师 $675$ 人；截至 $2022$ 年 $2$ 月，学校有教学班 $100$ 余个学生 $9000$ 余人。
